你需要的一切功能

为个人和团队打造的完整媒体托管解决方案。

拖拽上传

拖拽即可上传任何文件类型，图片自动生成缩略图，体验流畅。

即时分享链接

每个文件自动生成短链接，方便嵌入博客、文档或应用中分享。

相册管理

将文件整理到相册中，支持公开或私有权限，轻松管理资源。

API 令牌接入

生成独立的 API 令牌，无缝对接 PicGo、ShareX 等第三方上传工具。

多语言支持

内置国际化支持，中英文无缝切换，面向全球用户。

S3 兼容存储

支持本地存储、AWS S3、Cloudflare R2、MinIO 等 S3 兼容后端。

三步开始使用

从零到完整功能，只需几分钟。

1

创建账号

使用邮箱注册，或使用首次启动时生成的管理员账号登录。

2

上传文件

拖拽上传图片、视频、音频或文档，自动生成缩略图和访问链接。

3

分享与嵌入

复制生成的链接，在博客、文档或应用中使用，即贴即用。
